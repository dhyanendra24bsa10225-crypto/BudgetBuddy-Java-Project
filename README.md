# BudgetBuddy-Java-Project
BudgetBuddy is a local Java console app for secure finance tracking. It reads raw CSV data, uses OOP (Encapsulation, Polymorphism) and modular design to categorize transactions automatically. It generates summary reports calculating net savings/loss, demonstrating robust I/O and data processing.
SimpleProjectApp – Java Version

SimpleProjectApp is a modular Java application that demonstrates core programming concepts such as OOP, modularization, requirement analysis, and algorithmic implementation.

Features
•	User registration and authentication
•	Add numeric records with category tagging
•	Generate summary analytics (min, max, avg)
•	Category‑wise breakdown reports

How to Run
1. Ensure JDK 8 or higher is installed.
2. Compile the project using:
   javac *.java
3. Run the application using:
   java MainApp

Project Structure

•	User.java – Model for user entity
•	UserManager.java – Handles registration & login
•	Record.java – Model for numeric entries
•	DataManager.java – Manages stored records
•	ReportManager.java – Analytics and reporting
•	MainApp.java – Console interface

Notes
This project is built for academic demonstration and can be extended into a full‑scale application with persistent storage, GUI, and automated testing.
